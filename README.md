## 功能简介

---

* 用户登录
* 用户注册
* 浏览文章
* 发布文章
* 关注用户
* 评论文章
* 回复评论
* 修改用户信息

## 启动项目

```(command)
npm install
```

### 编译和热重载开发

```(cmd)
npm run serve
```

### Compiles and minifies for production

```(cmd)
npm run build
```

### 运行测试

```(cmd)
npm run test
```

### 检测和修复

```(cmd)
npm run lint
```
